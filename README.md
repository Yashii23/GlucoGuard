## 📌 Overview

GlucoGuard is an AI-powered healthcare platform that assists diabetic patients in monitoring their blood glucose, analyzing food intake, predicting sugar levels, and generating doctor-ready clinical reports.

The application combines modern web technologies with Google Gemini AI to provide intelligent health insights while maintaining an intuitive and responsive user experience.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Yashii23/GlucoGuard.git
```

## Navigate

```bash
cd GlucoGuard
```

## Install Dependencies

```bash
npm install
```

## Create Environment File

Create a `.env.local` file.

```env
GEMINI_API_KEY=YOUR_API_KEY
```

## Start Development Server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
